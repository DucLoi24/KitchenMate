## Ghi chú

- Tasks đánh dấu `*` là tùy chọn, có thể bỏ qua để triển khai MVP nhanh hơn
- Mỗi task tham chiếu đến yêu cầu cụ thể để đảm bảo traceability
- Property tests sử dụng thư viện `hypothesis` với `@settings(max_examples=100)`
- Tag format cho property tests: `# Feature: phase-4-api-endpoints, Property {N}: {mô tả}`
- Tất cả ViewSet dùng `IsOwner` từ `core/permissions.py` cho các thao tác ghi
- `IsAdminUser` trong `core/permissions.py` kiểm tra `is_staff=True`
