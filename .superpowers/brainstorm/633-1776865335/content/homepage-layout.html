Trang chủ - Layout chính

Chọn cách sắp xếp các section trên trang chủ

A

Hero + Sections liên tiếp

Hero banner ở trên → "Công thức mới nhất" → "Công thức phổ biến". Mỗi section là một grid 3-4 cột.

B

Hero + Tab navigation

Hero banner → 2 tabs: "Mới nhất" / "Phổ biến". Chuyển tab để xem danh sách khác nhau.

C

Hero + Infinite scroll

Hero banner → Danh sách recipe duy nhất, load thêm khi scroll xuống. Sort dropdown để chọn "Mới nhất" / "Phổ biến".

MÔ TẢ

A: Classic approach, show all at once. Good for discovery.

B: Space-saving, user picks view mode. Clean but less discovery.

C: Modern, mobile-friendly. Best for feed-like experience.
